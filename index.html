Flutter & NodeMCU ESP-32

Pendeteksi Kebocoran Gas Berbasis IoT

Kodingan

Project ini membuat suatu alat yang dapat mendeteksi kebocoran gas menggunakan sensor MQ-6, data yang dibaca oleh alat ini akan dihubungkan melalui firebase sehingga dapat dimonitoring. Data yang telah dihunbungkan ke firebase disalurkan lagi menuju aplikasi yang dibuat dengan flutter sehingga alat tersebut dapat di monitoring secara jarak jauh melalui aplikasi dan menggunakan internet. Untuk Mendapatkan aplikasi ini diperlukan install secara manual menggunakan kabel USB.
